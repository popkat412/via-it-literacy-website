1. 在社区中尽可能尊重他人，以您希望被对待的方式对待他人
2. 尽量不要发布个人信息，这可能很危险。
3. 脸书也不会保护您免受危险社区的侵害，因此请注意您加入的任何群组。
4. 仅将您在现实生活中认识的人添加为朋友，并通过更改设置和隐私页面来将您的个人资
   料保密。
